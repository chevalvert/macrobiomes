## Development

### Installation

```console
$ git clone https://github.com/chevalvert/macrobiomes
$ cd macrobiomes
$ yarn install
$ yarn start
```

### Usage

```console
$ yarn start
$ yarn build
```

### Deployment to production using GitHub releases
```console
$ yarn version
```
